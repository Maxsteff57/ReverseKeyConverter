# Reverse Hex Converter

GUI-приложение для конвертации hex-ключей путём перестановки групп по 4 байта.

Поддерживает любые форматы входных данных: с префиксами, суффиксами, через запятую, скобки, pipe — лишний текст распознаётся и игнорируется автоматически.

## Возможности

- Конвертация в реальном времени с двумя панелями
- Галочка **«Только hex-байты»** — убирает из результата всё лишнее
- Красная подсветка строк с некорректным количеством байт
- Зелёная подсветка групп по 4 байта при наведении
- Журнал ошибок с временными метками
- Горячие клавиши: `Ctrl+V` `Ctrl+S` `Ctrl+O` `Ctrl+D` `Ctrl+L`
- Drag & Drop загрузка файлов

## Поддерживаемые форматы

```
F8 BB 1E BE ...
07_56    D2 5D 86 42 ...
83 40 8A 5F, 83 40 8A 5F
```

## Требования

Windows 10/11 x64 — установка не требуется, просто запустите `.exe`.
